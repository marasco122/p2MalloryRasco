Answers for C:

Microwave Study Case

Design Issues

  • There are too many buttons, especially the extra features at the top.
  • The number pad should be the most obvious area of the control panel, but it
    takes up the same amount of space as the other features above it.
  • There's no way to backspace the previous input without deleting the whole line.
  • The button that controls the power is hidden among the extra features.
  • The timer button is in the corner and requires a little searching.
  • The clock goes away when the microwave is in use.

Answers for B:

Common Uses

  • The most common use of the microwave is to heat up some food for a certain amount
    of time.
  • There is usually a prominent clock on the device, that allows someone to easily and
    quickly check the time.
  • A timer is another common use of the microwave, when people need to time something
    else they are cooking in the kitchen.

This microwave does allow the user to operate it successfully, but there are still problems

when trying to move quickly. Mistakes happen very easily.

User Interactions

When a user wants to heat up some food:

  1. Place food in the microwave by opening the door.
  2. Set the timer using the number pad, inputing the numbers from left to right.
  3. Press Start to start cooking.
  4. When the timer ends, retrieve the food from microwave.

When a user wants to use the microwave as a timer:

  1. Press Timer button, so microwave does not heat up with nothing inside.
  2. Set the timer using the number pad, inputing the numbers from left to right.
  3. Press Start to start the timer.
  4. When the timer ends, press Cancel to end the alarm.

User Support

  • The buttons are all labeled as to what they mean, with words and symbols.
  • The numbers are all in the correct order.
  • There is a Cancel button, so the user can change their mind.

Feedback

  • The buttons press down a bit, for a physical response to being pressed.
  • There is also an audio response to the buttons, to confirm the input.
  • All the inputs are displayed on the screen, for the user to refer to.
  • There is an audio response and a visual display to the timer ending.

Common Mistakes

  • The wrong number is pressed, so the whole line must be deleted to be inputed again.
  • Not being able to find the Power button, so things are cooked at the wrong power.
  • Not pressing the Timer button, so the microwave runs while nothing is inside.

Design Improvements

  • The number pad will be bigger and more prominent, since they are the most used.
  • Most of the extra buttons will be removed entirely, as they are not completely
    necessary and just create more confusion and clutter.
  • There will be a backspace to remove the latest input.
  • The power level will change with a slider, so setting it will be simple. It will also
    be isolated on the panel and easier to find.
  • The timer button will be kept and displayed prominently, since it is still useful.
  • The current time will be displayed even when the microwave is in use, so users
    can always see it. The current date is also added to the display.
